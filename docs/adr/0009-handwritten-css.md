# ADR-0009: CSS — переносим как-есть из прототипа (CSS variables + утилитарные классы)

**Статус:** Accepted
**Дата:** 2026-05-06
**Принимающий:** platform team

## Контекст

Прототип использует ручную CSS с CSS-переменными и утилитарными классами (без Tailwind/styled-components). Стиль продуман и согласован.

## Альтернативы

- A: Переписать на Tailwind. Большая работа, потеря дизайна.
- B: Переписать на CSS Modules / Vanilla Extract. Средняя работа.
- C: Перенести как-есть, добавить Radix UI primitives для headless-компонентов (Dialog, Popover).

## Решение

C.

## Последствия

CSS-bundle ~22KB, простой, читаемый; компоненты Radix добавляются точечно.
