**Goal:** Deliver the reports module — six pre-defined report kinds (operator efficiency, project summary, calls-by-status, finance, quality control, hourly activity) + custom report endpoint, XLSX/CSV/PDF renderers, synchronous path for small windows, async path via `hibiken/asynq` for windows > 30 d OR estimated rows > 100 k, 24 h presigned-URL S3 download, audit-event publishing on every export, full RBAC + cross-tenant guard at the HTTP layer.

**Architecture:** Standard module shape `internal/reports/{api,service,store,events,transport,templates}`. Reports consumes the `analytics.ServiceRO` aggregate only (never reaches into `analytics/store` — depguard blocks it). Artifacts persist to a tenant-scoped S3 bucket via an extended `internal/recording/storage.ObjectStore` port (now exposing `Put` + `PresignedURL` alongside the existing `Get`/`Delete`). Job lifecycle persists to a new `reports_jobs` Postgres table protected by RLS; state transitions, audit-event emission, and `reports.report.ready` NATS publish all happen in the same `WithTenant` tx via `pkg/outbox`. Async path uses `hibiken/asynq` (already in deps; matches the crm + recording usage pattern).

**Tech Stack:** Go 1.26 · `github.com/xuri/excelize/v2 v2.10.1` (XLSX) · `github.com/signintech/gopdf` (NEW dep) · `encoding/csv` (CSV) · `github.com/hibiken/asynq v0.26.0` (async queue) · `gin-gonic/gin` (HTTP) · `jackc/pgx/v5` (Postgres) · `pkg/outbox` (durable cross-process events) · `pkg/middleware/tenant.RequireSameTenant` (tenant guard).

## Conventions and global rules

Read once. Apply per task. Repeat in commit messages where relevant.

- **Tenant scoping is mandatory.** Every `reports_jobs` query takes `tenant_id`; reads use `pool.WithTenant(ctx, tenantID, fn)` per RLS; cross-tenant lookups use `pool.BypassRLS` ONLY in `MarkSucceededTx` chain-of-custody flows when the consumer side already verified caller identity.
- **No floats in money or duration aggregates.** Counts are `uint64`, durations seconds are `uint64`. Floats live only in final-stage `float64` ratios computed at render time inside the template.
- **Time windows are half-open** — `[from, to)`. `from` inclusive, `to` exclusive. UI passes ISO timestamps; service validates via `analyticsapi.Window.Validate()`.
- **Async threshold** — `service.IsAsyncRequired(window, est)` returns `true` if `window.To.Sub(window.From) > AsyncPeriodDays*24h` OR `est > AsyncRowThreshold` OR `kind == api.KindCustom`. Defaults: 30 d / 100 000 rows.
- **Audit on every export** — `POST /api/reports/:kind/export`, `POST /api/reports/custom`, every `MarkSucceededTx`, every `GET /api/reports/jobs/:id/download` writes an audit-event to outbox `tenant.<t>.audit.event`. Payload conforms to `auditapi.Event` JSON tags. Subject is queued atomically with the row change via the same `outbox.Writer.Append`.
- **OpenTelemetry / Prometheus metrics** — `sociopulse_reports_jobs_total{tenant_id,kind,format,state}` (counter), `sociopulse_reports_render_duration_seconds{kind,format}` (histogram), `sociopulse_reports_artifact_bytes{kind,format}` (histogram), `sociopulse_reports_download_url_signed_total{tenant_id,kind}` (counter).
- **PDF row cap** — synchronous renderer returns `api.ErrTooLarge` when detail rows > 5 000 for PDF format. Caller switches to `format=xlsx` or `/api/reports/custom`.
- **Error envelope** — `{"code": "<stable_string>", "message": "<human_readable>"}` per `pkg/httputil/error_handler.go`. Stable codes: `reports.unknown_kind`, `reports.unsupported_format`, `reports.invalid_params`, `reports.job_not_found`, `reports.too_large`, `reports.canceled`, `reports.async_required`, `reports.window_invalid`.
- **Stable test fixture timezone:** `time.UTC`. Any time-bucket math in tests uses fixed UTC instants — never `time.Now()` directly in assertions.
- **`t.Parallel()` everywhere** the subtest doesn't share mutable state. Goleak across the package via `goleak.VerifyTestMain` in `package_test.go` files (per `golang-testing` skill).
- **Pre-commit gate per task** — every task ends with `make ci && go test -race -count=1 ./... && gofmt -l . && make grep-time-after` ALL green BEFORE `git commit`.
- **`git add` specific files only** — no `git add .` or `git add -A`. Specific paths shown per task.

```go
package storage

import (
	"bytes"
	"context"
	"errors"
	"fmt"
	"io"
	"net/url"
	"sync"
	"time"
)

// ObjectStore is the project-wide view of object storage. Production
// runs against Yandex Object Storage (Plan 01). Reports + recording
// share this port; if a 3rd module shows up, extract to pkg/objectstore.
type ObjectStore interface {
	// Get streams the object as ciphertext. The returned ReadCloser
	// MUST be closed by the caller.
	Get(ctx context.Context, bucket, key string) (io.ReadCloser, error)

	// Delete removes the object. Idempotent.
	Delete(ctx context.Context, bucket, key string) error

	// Put uploads payload under (bucket, key) with the given Content-Type.
	// Overwrites existing objects; matches S3 PUT semantics.
	Put(ctx context.Context, bucket, key string, payload []byte, contentType string) error

	// PresignedURL returns a time-limited URL granting GET access to
	// (bucket, key). The implementation MUST encode the TTL into the URL
	// (Yandex S3 uses the X-Amz-Expires query param). Production callers
	// pass 24*time.Hour (reports artifacts).
	PresignedURL(ctx context.Context, bucket, key string, ttl time.Duration) (string, error)
}

// ErrObjectNotFound is returned by Get when the object is missing.
var ErrObjectNotFound = errors.New("recording.storage: object not found")

// LocalObjectStore is an in-memory ObjectStore for tests and local dev.
// Concurrent-safe via sync.RWMutex.
//
// Production uses the Yandex S3 implementation (Plan 01).
type LocalObjectStore struct {
	mu      sync.RWMutex
	objects map[string]map[string]localBlob
}

type localBlob struct {
	payload     []byte
	contentType string
}

func NewLocalObjectStore() *LocalObjectStore {
	return &LocalObjectStore{objects: make(map[string]map[string]localBlob)}
}

var _ ObjectStore = (*LocalObjectStore)(nil)

func (s *LocalObjectStore) Get(ctx context.Context, bucket, key string) (io.ReadCloser, error) {
	if err := ctx.Err(); err != nil {
		return nil, err
	}
	s.mu.RLock()
	defer s.mu.RUnlock()
	b, ok := s.objects[bucket]
	if !ok {
		return nil, fmt.Errorf("%w: bucket=%s key=%s", ErrObjectNotFound, bucket, key)
	}
	blob, ok := b[key]
	if !ok {
		return nil, fmt.Errorf("%w: bucket=%s key=%s", ErrObjectNotFound, bucket, key)
	}
	return io.NopCloser(bytes.NewReader(bytes.Clone(blob.payload))), nil
}

func (s *LocalObjectStore) Delete(ctx context.Context, bucket, key string) error {
	if err := ctx.Err(); err != nil {
		return err
	}
	s.mu.Lock()
	defer s.mu.Unlock()
	if b, ok := s.objects[bucket]; ok {
		delete(b, key)
	}
	return nil
}

// Put stores the payload under (bucket, key) and overwrites any existing
// entry. The contentType field is held for symmetry with the production
// Yandex S3 implementation; the local store does not actually serve a
// Content-Type header.
func (s *LocalObjectStore) Put(ctx context.Context, bucket, key string, payload []byte, contentType string) error {
	if err := ctx.Err(); err != nil {
		return err
	}
	s.mu.Lock()
	defer s.mu.Unlock()
	if _, ok := s.objects[bucket]; !ok {
		s.objects[bucket] = make(map[string]localBlob)
	}
	s.objects[bucket][key] = localBlob{payload: bytes.Clone(payload), contentType: contentType}
	return nil
}

// PresignedURL returns a stub URL of the shape
//
//	local://<bucket>/<key>?expires=<unix>
//
// Production overrides this with the Yandex SDK presigner.
func (s *LocalObjectStore) PresignedURL(ctx context.Context, bucket, key string, ttl time.Duration) (string, error) {
	if err := ctx.Err(); err != nil {
		return "", err
	}
	if ttl <= 0 {
		return "", fmt.Errorf("recording.storage: ttl must be positive, got %v", ttl)
	}
	u := url.URL{
		Scheme: "local",
		Host:   bucket,
		Path:   "/" + key,
	}
	q := u.Query()
	q.Set("expires", fmt.Sprintf("%d", time.Now().UTC().Add(ttl).Unix()))
	u.RawQuery = q.Encode()
	return u.String(), nil
}

// PutBytes is the legacy test seam that pre-dates Put. Forwards to Put
// with an empty content type. Kept for backwards compatibility with
// existing recording tests; do NOT use in new code.
//
// Deprecated: use Put.
func (s *LocalObjectStore) PutBytes(bucket, key string, payload []byte) {
	_ = s.Put(context.Background(), bucket, key, payload, "")
}
```

```go
// Querier is the minimum surface PG needs (pgxpool.Pool + pgxmock both satisfy).
type Querier interface {
	Exec(ctx context.Context, sql string, args ...any) (pgconn.CommandTag, error)
	Query(ctx context.Context, sql string, args ...any) (pgx.Rows, error)
	QueryRow(ctx context.Context, sql string, args ...any) pgx.Row
}

type PG struct{ q Querier }

func NewPG(q Querier) *PG { return &PG{q: q} }

// CreateInput is the data needed to insert a new job row.
type CreateInput struct {
	ID           string
	TenantID     uuid.UUID
	Kind         reportsapi.ReportKind
	Format       reportsapi.ExportFormat
	Params       map[string]any
	WindowFrom   time.Time
	WindowTo     time.Time
	CreatedBy    uuid.UUID
	NotifyUserID uuid.UUID
}

const insertJobSQL = `
INSERT INTO reports_jobs
    (id, tenant_id, kind, format, params, window_from, window_to, created_by, notify_user_id)
VALUES
    ($1, $2, $3, $4, $5, $6, $7, $8, $9)
`

func (s *PG) Create(ctx context.Context, in CreateInput) error {
	paramsJSON, err := json.Marshal(in.Params)
	if err != nil {
		return fmt.Errorf("reports.store.Create: marshal params: %w", err)
	}
	if _, err := s.q.Exec(ctx, insertJobSQL,
		in.ID, in.TenantID, string(in.Kind), string(in.Format),
		paramsJSON, in.WindowFrom, in.WindowTo, in.CreatedBy, in.NotifyUserID,
	); err != nil {
		return fmt.Errorf("reports.store.Create: exec: %w", err)
	}
	return nil
}

const selectJobSQL = `
SELECT id, tenant_id, kind, format, params, window_from, window_to,
       state, started_at, finished_at, bytes_size, filename, download_url,
       error, created_by, notify_user_id, created_at
FROM reports_jobs
WHERE id = $1
`

func (s *PG) Get(ctx context.Context, jobID string) (reportsapi.Job, error) {
	row := s.q.QueryRow(ctx, selectJobSQL, jobID)
	var (
		params  []byte
		state, kind, format, filename, dlURL, errStr string
		started, finished *time.Time
		job reportsapi.Job
	)
	err := row.Scan(
		&job.ID, &job.TenantID, &kind, &format, &params,
		&job.Window.From, &job.Window.To,
		&state, &started, &finished, &job.BytesSize, &filename, &dlURL, &errStr,
		&job.CreatedBy, &job.NotifyUserID, &job.CreatedAt,
	)
	if err != nil {
		if errors.Is(err, pgx.ErrNoRows) || strings.Contains(err.Error(), "no rows") {
			return reportsapi.Job{}, fmt.Errorf("%w: %s", reportsapi.ErrJobNotFound, jobID)
		}
		return reportsapi.Job{}, fmt.Errorf("reports.store.Get: scan: %w", err)
	}
	if len(params) > 0 {

	}
	job.Kind = reportsapi.ReportKind(kind)
	job.Format = reportsapi.ExportFormat(format)
	job.State = reportsapi.JobState(state)
	job.StartedAt = started
	job.FinishedAt = finished
	job.Filename = filename
	job.DownloadURL = dlURL
	job.Error = errStr
	_ = analyticsapi.Window{} // imported only for documentation of Window provenance
	return job, nil
}
```

```go
		job.State = reportsapi.JobState(state)
		job.StartedAt, job.FinishedAt = started, finished

		last := out[len(out)-1]
		nextCursor = encodeCursor(last.CreatedAt, last.ID)
```

```go
package store

import (
	"context"
	"errors"
	"fmt"
	"time"

	"github.com/jackc/pgx/v5"
)

// TxQuerier is the subset of pgx.Tx the *Tx variants need.
type TxQuerier interface {
	Exec(ctx context.Context, sql string, args ...any) (interface{ RowsAffected() int64 }, error)
}

// ErrStaleSkip is returned by *Tx state-flip methods when the underlying
// UPDATE affected zero rows — the row was already in a terminal state OR
// belongs to another tenant. Callers ack the asynq task and skip audit
// emission per the errStaleSkip pattern (Plan 12.4).
var ErrStaleSkip = errors.New("reports.store: stale state-flip skipped")

const flipRunningSQL = `
UPDATE reports_jobs
SET state = 'running', started_at = $2
WHERE id = $1 AND state = 'queued'
`

func MarkRunningTx(ctx context.Context, tx pgx.Tx, jobID string, ts time.Time) error {
	tag, err := tx.Exec(ctx, flipRunningSQL, jobID, ts)
	if err != nil {
		return fmt.Errorf("reports.store.MarkRunningTx: %w", err)
	}
	if tag.RowsAffected() == 0 {
		return ErrStaleSkip
	}
	return nil
}

const flipSucceededSQL = `
UPDATE reports_jobs
SET state = 'succeeded', finished_at = $2,
    bytes_size = $3, filename = $4, download_url = $5
WHERE id = $1 AND state IN ('queued', 'running')
`

func MarkSucceededTx(
	ctx context.Context, tx pgx.Tx, jobID string, ts time.Time,
	bytesSize int64, filename, presignedURL string,
) error {
	tag, err := tx.Exec(ctx, flipSucceededSQL, jobID, ts, bytesSize, filename, presignedURL)
	if err != nil {
		return fmt.Errorf("reports.store.MarkSucceededTx: %w", err)
	}
	if tag.RowsAffected() == 0 {
		return ErrStaleSkip
	}
	return nil
}

const flipFailedSQL = `
UPDATE reports_jobs
SET state = 'failed', finished_at = $2, error = $3
WHERE id = $1 AND state IN ('queued', 'running')
`

func MarkFailedTx(ctx context.Context, tx pgx.Tx, jobID string, ts time.Time, errMsg string) error {
	tag, err := tx.Exec(ctx, flipFailedSQL, jobID, ts, errMsg)
	if err != nil {
		return fmt.Errorf("reports.store.MarkFailedTx: %w", err)
	}
	if tag.RowsAffected() == 0 {
		return ErrStaleSkip
	}
	return nil
}

const flipCanceledSQL = `
UPDATE reports_jobs
SET state = 'canceled', finished_at = $2, error = 'canceled by caller'
WHERE id = $1 AND state IN ('queued', 'running')
`

func MarkCanceledTx(ctx context.Context, tx pgx.Tx, jobID string, ts time.Time) error {
	tag, err := tx.Exec(ctx, flipCanceledSQL, jobID, ts)
	if err != nil {
		return fmt.Errorf("reports.store.MarkCanceledTx: %w", err)
	}
	if tag.RowsAffected() == 0 {
		return ErrStaleSkip
	}
	return nil
}
```

```go
// audit Service (Plan 03 Task 7) subscribes and persists; until then
// the event sits durably in the outbox table awaiting a consumer.

	actorID := in.ActorID

			"job_id":       in.JobID,
			"kind":         in.Kind,
			"format":       in.Format,
			"bytes_size":   in.BytesSize,
			"window_from":  in.Window.From,
			"window_to":    in.Window.To,
			"params":       in.Params,
		},
```

```go
func paramUUID(p map[string]any, key string) (uuid.UUID, error) {
	raw, ok := p[key].(string)
	if !ok {
		return uuid.UUID{}, fmt.Errorf("%w: missing %s", reportsapi.ErrInvalidParams, key)
	}
	return uuid.Parse(raw)
}

func paramUUIDOpt(p map[string]any, key string) (uuid.UUID, error) {
	raw, ok := p[key].(string)
	if !ok || raw == "" {
		return uuid.Nil, nil
	}
	return uuid.Parse(raw)
}

func paramFloat(p map[string]any, key string) (float64, error) {
	switch v := p[key].(type) {
	case float64:
		return v, nil
	case int:
		return float64(v), nil
	case nil:
		return 0, fmt.Errorf("%w: missing %s", reportsapi.ErrInvalidParams, key)
	default:
		return 0, fmt.Errorf("%w: %s is not numeric", reportsapi.ErrInvalidParams, key)
	}
}
```

```go
	return pdf.Cell(nil, title)

// PDFRow writes one row of N cells starting at y. Returns the new y after.
func PDFRow(pdf *gopdf.GoPdf, y float64, cells []string, widths []float64) (float64, error) {
	x := 40.0
	for i, c := range cells {
		pdf.SetXY(x, y)
		if err := pdf.Cell(nil, c); err != nil {
			return 0, err
		}
		x += widths[i]
	}
	return y + 18, nil
}
```

```go
// RunnerDeps is the set of ports the runner needs.
type RunnerDeps struct {
	Analytics    analyticsapi.ServiceRO
	Store        *reportstore.PG
	Pool         *postgres.Pool        // for WithTenant Tx
	ObjectStore  storage.ObjectStore
	Audit        *AuditEmitter
	ReadyPub     *rptevents.ReportReadyPublisher
	BucketPrefix string                // "sociopulse-reports"
	PresignTTL   time.Duration         // 24h
	Threshold    ThresholdConfig
	Now          func() time.Time      // injectable for tests
}

type Runner struct{ deps RunnerDeps }

func NewRunner(d RunnerDeps) *Runner {
	if d.Now == nil {
		d.Now = time.Now
	}
	return &Runner{deps: d}
}

var _ reportsapi.ReportRunner = (*Runner)(nil)

// Run executes a synchronous report. Returns ErrAsyncRequired if the
// caller's window/estimate trips the threshold; ErrTooLarge if a renderer
// rejects (e.g. PDF >5000 rows).
func (r *Runner) Run(ctx context.Context, in reportsapi.RunInput) (reportsapi.RunResult, error) {
	if err := validateInput(in); err != nil {
		return reportsapi.RunResult{}, err
	}
	if IsAsyncRequired(r.deps.Threshold, in.Window, estimateRows(in.Kind), in.Kind) {
		return reportsapi.RunResult{}, fmt.Errorf("%w: switch to async path", reportsapi.ErrUnsupportedFmt)
	}
	res, err := renderByKind(ctx, r.deps.Analytics, in)
	if err != nil {
		return reportsapi.RunResult{}, err
	}
	// Sync path: no persistent reports_jobs row; upload to S3 + sign URL,
	// emit audit-event via outbox (using a one-row Tx).
	bucket := fmt.Sprintf("%s-%s", r.deps.BucketPrefix, in.TenantID)
	key := syntheticKey(in.Kind, in.ActorID, res.Filename, r.deps.Now())
	if err := r.deps.ObjectStore.Put(ctx, bucket, key, res.Bytes, res.MIME); err != nil {
		return reportsapi.RunResult{}, fmt.Errorf("runner.Run: object Put: %w", err)
	}
	presigned, err := r.deps.ObjectStore.PresignedURL(ctx, bucket, key, r.deps.PresignTTL)
	if err != nil {
		return reportsapi.RunResult{}, fmt.Errorf("runner.Run: presign: %w", err)
	}
	if err := r.deps.Pool.WithTenant(ctx, in.TenantID, func(tctx context.Context) error {
		tx, err := r.deps.Pool.Begin(tctx)
		if err != nil {
			return err
		}
		defer tx.Rollback(context.Background()) //nolint:errcheck
		if err := r.deps.Audit.EmitTx(tctx, tx, AuditExport{
			TenantID: in.TenantID, ActorID: in.ActorID, Action: auditapi.ActorUser,
			JobID: fmt.Sprintf("sync:%s", key), Kind: string(in.Kind), Format: string(in.Format),
			BytesSize: int64(len(res.Bytes)), Window: struct{ From, To time.Time }{
				From: in.Window.From, To: in.Window.To,
			}, Params: in.Params, Timestamp: r.deps.Now().UTC(),
		}); err != nil {
			return err
		}
		return tx.Commit(tctx)
	}); err != nil {
		return reportsapi.RunResult{}, err
	}
	res.Filename = res.Filename // unchanged; URL returned to caller separately
	_ = presigned // sync path returns bytes directly; download URL appears in the HTTP response envelope, see Task 7
	return res, nil
}

func validateInput(in reportsapi.RunInput) error {
```

```go
import (
	"context"
	"encoding/json"
	"fmt"
	"time"

	"github.com/google/uuid"
	"github.com/hibiken/asynq"
	"github.com/jackc/pgx/v5"

	reportsapi "github.com/sociopulse/platform/internal/reports/api"
	reportstore "github.com/sociopulse/platform/internal/reports/store"
	"github.com/sociopulse/platform/pkg/postgres"
)

// AsynqEnqueuer is the asynq.Client surface the queue needs (test seam).
type AsynqEnqueuer interface {
	EnqueueContext(ctx context.Context, task *asynq.Task, opts ...asynq.Option) (*asynq.TaskInfo, error)
}

type Queue struct {
	store    *reportstore.PG
	pool     *postgres.Pool
	enq      AsynqEnqueuer
	queueName string
}

func NewQueue(store *reportstore.PG, pool *postgres.Pool, enq AsynqEnqueuer, queueName string) *Queue {
	if queueName == "" {
		queueName = "reports"
	}
	return &Queue{store: store, pool: pool, enq: enq, queueName: queueName}
}

var _ reportsapi.JobQueue = (*Queue)(nil)

func (q *Queue) Enqueue(ctx context.Context, in reportsapi.JobInput) (reportsapi.JobTicket, error) {
	if err := in.Window.Validate(); err != nil {
		return reportsapi.JobTicket{}, err
	}
	if in.TenantID == uuid.Nil {
		return reportsapi.JobTicket{}, fmt.Errorf("queue.Enqueue: %w: tenant_id missing", reportsapi.ErrInvalidParams)
	}

	payload, err := json.Marshal(in)
	if err != nil {
		return reportsapi.JobTicket{}, fmt.Errorf("queue.Enqueue: marshal: %w", err)
	}
	task := asynq.NewTask(reportsapi.TaskJobRun, payload)

	var info *asynq.TaskInfo
	if err := q.pool.WithTenant(ctx, in.TenantID, func(tctx context.Context) error {
		// Enqueue first to get the task ID; persist using that ID.
		var enqErr error
		info, enqErr = q.enq.EnqueueContext(tctx, task, asynq.Queue(q.queueName))
		if enqErr != nil {
			return fmt.Errorf("queue.Enqueue: asynq: %w", enqErr)
		}
		return q.store.Create(tctx, reportstore.CreateInput{
			ID: info.ID, TenantID: in.TenantID,
			Kind: in.Kind, Format: in.Format,
			Params: in.Params, WindowFrom: in.Window.From, WindowTo: in.Window.To,
			CreatedBy: in.ActorID, NotifyUserID: in.NotifyUserID,
		})
	}); err != nil {
		return reportsapi.JobTicket{}, err
	}
	return reportsapi.JobTicket{JobID: info.ID, QueuedAt: time.Now().UTC()}, nil
}

func (q *Queue) Get(ctx context.Context, jobID string) (reportsapi.Job, error) {
	var job reportsapi.Job
	err := q.pool.BypassRLS(ctx, func(c context.Context) error {
		var err error
		job, err = q.store.Get(c, jobID)
		return err
	})
	return job, err
}

func (q *Queue) List(ctx context.Context, f reportsapi.ListJobsFilter) ([]reportsapi.Job, string, error) {
	var rows []reportsapi.Job
	var cursor string
	err := q.pool.WithTenantFromCtx(ctx, func(c context.Context) error {
		var err error
		rows, cursor, err = q.store.List(c, f)
		return err
	})
	return rows, cursor, err
}

func (q *Queue) Cancel(ctx context.Context, jobID string) error {
	job, err := q.Get(ctx, jobID)
	if err != nil {
		return err
	}
	return q.pool.WithTenant(ctx, job.TenantID, func(tctx context.Context) error {
		tx, err := q.pool.Begin(tctx)
		if err != nil {
			return err
		}
		defer tx.Rollback(context.Background()) //nolint:errcheck
		if err := reportstore.MarkCanceledTx(tctx, tx, jobID, time.Now().UTC()); err != nil {
			if err == reportstore.ErrStaleSkip {
				return nil // idempotent
			}
			return err
		}
		return tx.Commit(tctx)
	})
}
```

```go
	"context"
	"encoding/json"

type ConsumerDeps struct {
	Server       *asynq.Server
	Mux          *asynq.ServeMux
	Analytics    analyticsapi.ServiceRO
	Store        *reportstore.PG
	Pool         *postgres.Pool
	ObjectStore  storage.ObjectStore
	Audit        *AuditEmitter
	ReadyPub     *rptevents.ReportReadyPublisher
	BucketPrefix string
	PresignTTL   time.Duration
	Logger       *zap.Logger
	Now          func() time.Time
}

type Consumer struct{ d ConsumerDeps }

func NewConsumer(d ConsumerDeps) *Consumer {
	if d.Now == nil {
		d.Now = time.Now
	}
	return &Consumer{d: d}
}

var _ reportsapi.JobConsumer = (*Consumer)(nil)

// Run wires the asynq handler and starts the server. Blocks until ctx cancels.
func (c *Consumer) Run(ctx context.Context) error {
	c.d.Mux.HandleFunc(reportsapi.TaskJobRun, c.handleJobRun)

	errCh := make(chan error, 1)
	go func() { errCh <- c.d.Server.Run(c.d.Mux) }()
	select {
	case <-ctx.Done():
		c.d.Server.Shutdown()
		<-errCh
		return nil
	case err := <-errCh:
		return err
	}
}

func (c *Consumer) handleJobRun(ctx context.Context, task *asynq.Task) error {
	var in reportsapi.JobInput
	if err := json.Unmarshal(task.Payload(), &in); err != nil {
		c.d.Logger.Error("reports.consumer: bad payload", zap.Error(err))
		return fmt.Errorf("%w: %v", asynq.SkipRetry, err)
	}
	jobID, _ := asynq.GetTaskID(ctx)
	tenantID := in.TenantID
	now := c.d.Now().UTC()

	// MarkRunning (idempotent — ErrStaleSkip just acks).
	if err := c.d.Pool.WithTenant(ctx, tenantID, func(tctx context.Context) error {
		tx, err := c.d.Pool.Begin(tctx)
		if err != nil {
			return err
		}
		defer tx.Rollback(context.Background()) //nolint:errcheck
		if err := reportstore.MarkRunningTx(tctx, tx, jobID, now); err != nil {
			if errors.Is(err, reportstore.ErrStaleSkip) {
				return nil
			}
			return err
		}
		return tx.Commit(tctx)
	}); err != nil {
		c.d.Logger.Error("reports.consumer: MarkRunning failed", zap.String("job", jobID), zap.Error(err))
		return err
	}

	// Render.
	res, renderErr := renderByKind(ctx, c.d.Analytics, in.RenderInput)
	if renderErr != nil {
		return c.markFailed(ctx, tenantID, in, jobID, renderErr)
	}

	// Upload + sign.
	bucket := fmt.Sprintf("%s-%s", c.d.BucketPrefix, tenantID)
	key := syntheticKey(in.Kind, in.ActorID, res.Filename, now)
	if err := c.d.ObjectStore.Put(ctx, bucket, key, res.Bytes, res.MIME); err != nil {
		return c.markFailed(ctx, tenantID, in, jobID, fmt.Errorf("upload: %w", err))
	}
	url, err := c.d.ObjectStore.PresignedURL(ctx, bucket, key, c.d.PresignTTL)
	if err != nil {
		return c.markFailed(ctx, tenantID, in, jobID, fmt.Errorf("presign: %w", err))
	}

	// MarkSucceeded + audit-emit + report-ready in one tx.
	return c.d.Pool.WithTenant(ctx, tenantID, func(tctx context.Context) error {
		tx, err := c.d.Pool.Begin(tctx)
		if err != nil {
			return err
		}
		defer tx.Rollback(context.Background()) //nolint:errcheck

		if err := reportstore.MarkSucceededTx(tctx, tx, jobID, c.d.Now().UTC(),
			int64(len(res.Bytes)), res.Filename, url); err != nil {
			if errors.Is(err, reportstore.ErrStaleSkip) {
				return nil // idempotent
			}
			return err
		}
		if err := c.d.Audit.EmitTx(tctx, tx, AuditExport{
			TenantID: tenantID, ActorID: in.ActorID, Action: auditapi.ActorUser,
			JobID: jobID, Kind: string(in.Kind), Format: string(in.Format),
			BytesSize: int64(len(res.Bytes)),
			Window: struct{ From, To time.Time }{From: in.Window.From, To: in.Window.To},
			Params: in.Params, Timestamp: c.d.Now().UTC(),
		}); err != nil {
			return err
		}
		if err := c.d.ReadyPub.PublishReadyTx(tctx, tx, tenantID, jobID, reportsapi.ReportReadyEvent{
			JobID: jobID, TenantID: tenantID.String(), Kind: string(in.Kind),
			Format: string(in.Format), Filename: res.Filename,
			BytesSize: int64(len(res.Bytes)), DownloadURL: url,
		}); err != nil {
			return err
		}
		return tx.Commit(tctx)
	})
}

func (c *Consumer) markFailed(
	ctx context.Context, tenantID interface{ String() string }, in reportsapi.JobInput, jobID string, cause error,
) error {
	msg := cause.Error()
	c.d.Logger.Error("reports.consumer: render failed", zap.String("job", jobID), zap.Error(cause))
	flipErr := c.d.Pool.WithTenant(ctx, tenantID.(any).(interface{}).(uuid.UUID), func(tctx context.Context) error {
		tx, err := c.d.Pool.Begin(tctx)
		if err != nil {
			return err
		}
		defer tx.Rollback(context.Background()) //nolint:errcheck
		if err := reportstore.MarkFailedTx(tctx, tx, jobID, c.d.Now().UTC(), msg); err != nil {
			if errors.Is(err, reportstore.ErrStaleSkip) {
				return nil
			}
			return err
		}
		if err := c.d.Audit.EmitTx(tctx, tx, AuditExport{
			TenantID: tenantID.(any).(uuid.UUID), ActorID: in.ActorID, Action: auditapi.ActorUser,
			JobID: jobID, Kind: string(in.Kind), Format: string(in.Format),
			Window: struct{ From, To time.Time }{From: in.Window.From, To: in.Window.To},
			Params: in.Params, Timestamp: c.d.Now().UTC(),
		}); err != nil {
			return err
		}
		return tx.Commit(tctx)
	})
	if flipErr != nil {
		c.d.Logger.Error("reports.consumer: MarkFailed flip failed", zap.String("job", jobID), zap.Error(flipErr))
	}
	// Permanent failures (bad params, unknown kind, unsupported fmt) SkipRetry.
	if errors.Is(cause, reportsapi.ErrInvalidParams) || errors.Is(cause, reportsapi.ErrUnknownKind) || errors.Is(cause, reportsapi.ErrUnsupportedFmt) || errors.Is(cause, reportsapi.ErrTooLarge) {
		return fmt.Errorf("%w: %v", asynq.SkipRetry, cause)
	}
	return cause // transient: asynq retries
}
```

```go
import (
```

```go
	"github.com/sociopulse/platform/internal/modules"
	"github.com/sociopulse/platform/pkg/outbox"
)

type Module struct{}

func (Module) Name() string { return "reports" }

func (Module) Register(d modules.Deps) error {
	ana, _ := modules.LocatorGet[analyticsapi.ServiceRO](d.Locator, "analytics.ServiceRO")
	if ana == nil {
		// analytics not registered → reports degrades to "list kinds" only;
		// log a warning and skip routes.
		d.Logger.Warn("reports.Module.Register: analytics.ServiceRO not in locator; skipping routes")
		return nil
	}
	objectStore, _ := modules.LocatorGet[storage.ObjectStore](d.Locator, "storage.ObjectStore")
	if objectStore == nil {
		d.Logger.Warn("reports.Module.Register: storage.ObjectStore not in locator; skipping routes")
		return nil
	}
	outboxWriter, _ := modules.LocatorGet[outbox.Writer](d.Locator, "outbox.Writer")
	if outboxWriter == nil {
		return nil
	}

	cfg := d.Config.Reports
	store := reportstore.NewPG(d.Pool)
	asynqClient := asynq.NewClient(asynq.RedisClientOpt{Addr: d.Config.Redis.Addr})
	audit := rptservice.NewAuditEmitter(outboxWriter)
	readyPub := rptevents.NewReportReadyPublisher(outboxWriter)

	svc := rptservice.Build(ana, store, d.Pool, objectStore, audit, readyPub, rptservice.Config{
		BucketPrefix: cfg.BucketPrefix, PresignTTL: cfg.PresignTTL,
		Threshold: rptservice.ThresholdConfig{
			AsyncPeriodDays: cfg.AsyncPeriodDays, AsyncRowThreshold: cfg.AsyncRowThreshold,
		},
	})
	svc.Queue = rptservice.NewQueue(store, d.Pool, asynqClient, cfg.QueueName)

	// HTTP — admin-only.
	requireAdmin := /* engineer: existing helper from auth transport, or inline */ nil
	rpttransport.Register(d.HTTPRouter, rpttransport.NewHandlers(svc), nil /* RBAC checker */, store, d.Pool, requireAdmin)

	// Locator registrations for the worker (cmd/worker constructs the Consumer).
	d.Locator.Set("reports.Service", svc)
	d.Locator.Set("reports.JobQueue", svc.Queue)
	_ = reportsapi.SubjectReportReadyFor // keep import alive
	_ = context.Background
	_ = time.Now
	return nil
}
```
